# Изменения в сайте Dizcorp

## Выполненные изменения:

### 1. Изменение курсора в разделе "Наши турниры"
- Изменен цвет стрелок курсора с белого на розовый (rgba(255, 105, 180, 0.8))
- Добавлен эффект размытия (blur(0.5px)) для стрелок
- Изменен цвет фона курсора на розовую стилистику сайта
- Обновлен цвет курсора в состоянии grabbing

### 2. Добавлена новая карточка турнира
- В разделе "НАША ЦЕЛЬ — СОЗДАВАТЬ НЕЗАБЫВАЕМЫЕ КИБЕРСПОРТИВНЫЕ СОБЫТИЯ"
- Заменена пустая карточка на большую карточку "Mobile Esports Fest" (3-й ряд, занимает 2 колонки)
- Добавлена маленькая карточка "Volt Energy Cup" в том же 3-м ряду справа с выравниванием по нижнему краю (align-self: end)
- Нижние границы большой и маленькой карточек теперь на одном уровне
- Использованы изображения mobile-fest.jpg и volt-energy-cup.jpg
- Обновлены CSS стили для корректного отображения 6 карточек в сетке из 3 рядов
- Добавлены адаптивные стили для мобильных устройств

### 3. Перемещение wall.gif
- Перемещен wall.gif на 50px выше от текущей позиции
- Обновлены все адаптивные стили для разных размеров экрана
- Изменения применены для всех медиа-запросов (1366px, 768px, 480px)

### 4. Добавлены центральные элементы
- Добавлен ct.gif по центру сайта (абсолютное позиционирование) - увеличен в 3 раза (240px)
- CS2.png перемещен выше на 1200px и правее на 640px с минимальной плавающей анимацией - увеличен в 2 раза (200px)
- Добавлен pers.gif (84px) - уменьшен на 30%, перемещен вверх на 100px и правее на 320px с плавающей анимацией
- Добавлен error.gif (100px) - правее центра, вверху
- Добавлен niz.png (80px) - по центру, внизу
- Добавлен verh.png (189px) - уменьшен на 30%, перемещен выше на 1130px и правее на 80px
- Элементы двигаются вместе с контентом при скролле (position: absolute)
- Элементы имеют полупрозрачность и не мешают взаимодействию с сайтом
- Добавлены адаптивные стили для всех размеров экрана с пропорциональным масштабированием
- Элементы позиционированы с разными z-index для корректного отображения

### 5. Добавлены анимации
- CS2.png имеет минимальную плавающую анимацию (6s, ±1° поворот, ±1% масштаб, ±2% движение)
- pers.gif имеет плавающую анимацию (4s, небольшие движения вверх-вниз и влево-вправо на ±3-4px)

## Файлы, которые были изменены:
- `/css/style.css` - стили курсора и расположение карточек турниров
- `/css/examp-effects.css` - позиция wall.gif, стили для всех центральных элементов, анимации CS2.png и pers.gif
- `/index.html` - добавлена новая карточка турнира и все центральные элементы (ct.gif, CS2.png, pers.gif, error.gif, niz.png, verh.png)
